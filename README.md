# 球探推荐——体育游泳者必备

  应用介绍:"球探推荐——体育游泳者必备"app是一款线上购买游泳装备和预约游泳课程的应用。
  
  应用功能:
  
    a.商城：泳装、泳镜、游泳装备；
      购买流程:用户下单后需用户持取货码到店取货，并且在店内付款取货。
      
    b.游泳课程：奇迹健身-恒温游泳俱乐部，针对儿童推出的游泳课程；
      预约流程:预约某游泳课程后需在预约的时间内到达游泳场馆进行使用。
    
    c.订单：商品订单、课程订单等两类。
      
    d.我的：个人信息更改、商品收藏、课程收藏、优惠券、设置。
